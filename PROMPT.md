# Daily News Collector - Notion MCP 用プロンプト

毎朝8:00に以下を実行してください。

## 共通ルール
- Web検索で**本日の日本語ニュース**を収集する
- 各DBに3〜5件のページを追加する
- 昨日以前の古いニュースは除外する
- 要約は2〜3文の日本語で、技術的なポイントが伝わるように書く
- URLは検索結果から取得した実在URLのみ使用（推測で生成しない）
- 同一ニュースの重複追加を避けるため、追加前にDB内の既存ページタイトルを確認する

## 各ページの共通プロパティ
| プロパティ名 | 型 | 内容 |
|---|---|---|
| タイトル | title | ニュースの見出し |
| 要約 | rich_text | 2〜3文の日本語要約 |
| ソースURL | url | 記事の実在URL |
| カテゴリタグ | multi_select | 関連技術・トピック（例: React, Next.js, OpenAI） |
| 重要度 | select | 高 / 中 / 低（業界へのインパクトで判断） |
| 収集日 | date | 収集実行日（ISO 8601） |
| ステータス | select | デフォルト「未読」で作成 |

## カテゴリ別設定

### 1. フロントエンド開発ニュース
- Database ID: 339fb2f3-7f0b-804a-8405-c467b662a8ca
- 検索クエリ: "Angular OR TypeScript OR React OR Next.js OR CSS OR Web標準 最新ニュース"
- 対象: フレームワークのリリース、ブラウザAPI更新、ツールチェーン変更など
- カテゴリタグ例: React, Next.js, Angular, TypeScript, CSS, Vite, Node.js

### 2. AI・LLM関連ニュース
- Database ID: 339fb2f3-7f0b-8076-8477-fb4e8d9d3b1b
- 検索クエリ: "AI OR LLM OR 生成AI OR Claude OR GPT OR Gemini 最新ニュース"
- 対象: モデルリリース、API更新、研究論文、規制動向など
- カテゴリタグ例: Claude, GPT, Gemini, LLM, 画像生成, エージェント, 規制

### 3. 一般ニュース
- Database ID: 339fb2f3-7f0b-8026-895d-e9800858fd99
- 検索クエリ: "日本 世界 主要ニュース 今日"
- 対象: 政治、経済、国際、社会の主要トピック
- カテゴリタグ例: 政治, 経済, 国際, 社会, テクノロジー

## 実行手順
1. 各カテゴリのDBから既存ページタイトルを取得（重複チェック用）
2. Web検索でニュースを収集
3. 重複を除外し、Notion MCPで各DBにページを追加
4. 追加完了後、カテゴリごとの追加件数を報告
